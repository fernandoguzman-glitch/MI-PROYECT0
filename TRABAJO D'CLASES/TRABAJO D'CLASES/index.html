HMTL5

HTML, siglas de HyperText Markup Language, es un lenguaje de marcado utilizado para estructurar y presentar contenido en páginas web. No es un lenguaje de programación, sino un conjunto de elementos que definen cómo se organiza y muestra el contenido, como texto, imágenes, listas y enlaces.

CSS3

CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje utilizado en programación web para definir la apariencia y el diseño visual de las páginas web. Mientras que HTML se encarga de estructurar el contenido de una página, CSS permite personalizar su presentación, como colores, fuentes, tamaños, márgenes, posiciones y otros aspectos estéticos.CSS funciona separando el contenido (HTML) de la presentación (CSS) , lo que facilita la mantenibilidad y modularidad del código. Esto significa que puedes tener un único archivo CSS que controle el diseño de múltiples páginas HTML , simplificando los cambios y reduciendo la duplicación de estilos.
